Utilities

  • pickacolor

    Service for Mac OS X that you can assign to a hotkey which will pop up a colorpicker. You choose a color and the hex digits go onto the clipboard.

  • Vimini.app

    Open files in vim in iTerm using Finder. Customize which iTerm profile to use. Also available from context menu.

  • tabcolor

    bash script: Color your iTerm tabs via command-line. Accepts an hex-color argument. Make the chrome of the window match the contents. Solarize all the way, baby!

  • Toggle Dock Lock

    Simple app for you to place on your dock. Use it when you want to lock or unlock the dock.

Blender Addons

  • mkaddon

    Blender addon: Boilerplate maker for blender addons. Will not write your addon for you, but may help you get started quicker.

  • Icons!

    Blender addon: Similar to the addon that comes with blender but involving fewer steps to produce a display of icons with associated names.

  • gradience

    Blender addon: Random sequence of colors to a ramp.

  • packcygon

    Blender addon: Select a bunch of things and apply a 2d bin-packing algorithm to them to fit them in a small space.

  • shiftsemi

    Blender addon: Press shift+semicolon in the viewport to rename the selection. Works on multiple objects. Renames data (also bones).

  • tkit

    Blender addon: Adds various ways of selecting neighboring mesh elements. Available in edge mode.

  • Particles to Point Cloud

    Blender addon: With a mesh object selected, rip the locations of the active particle system into a new mesh.
